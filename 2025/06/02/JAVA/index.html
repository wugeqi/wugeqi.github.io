JAVA

1.JAVA基础

（1）jvm、jre、jdk：

jvm：java虚拟机

jre:程序运行的的最小环境

jdk：一系列工具

面向对象：将世界上的一切东西当作对象，程序功能的实现就是通过对象的交互实现的

继承：指一个类可以实现另一个类中的所有对象和方法

封装：指对象不对外暴露细节，只对外界提供交互的接口

多态：

运行时多态：重写，指子类可以重写父类的方法

编译时多态：重载，指在同一个类中允许参数不同的重名方法

反射：运行时可以获取任意一个类的所有信息，字段，方法等

获取.class：1.根据完整包名2.根据实例的对象获得3.根据

动态代理：运行时动态生成代理

方法：

jdk：主要是对实现接口的类，通过调用invoktionhander，并重写invoke（）方法

cglib：主要是对没有实现接口的类，通过创建子类继承父类，然后对方法进行重写，实现动态代理

注解：根据字段决定程序的执行的方式

生命周期：

字节：编译时没有

编译：运行时没有

运行：都有

方法：反射获取字段，然后根据动态代理对原方法进行扩展

深拷贝和浅拷贝：

浅拷贝：不会递归拷贝引用，，也就是说如果是个引用对象，进行浅拷贝的时候，会创建一个新对象存储着原来的引用，也就是说两个对象指向着同一个引用

深拷贝：会递归拷贝引用

递归调用

clone

序列化

修饰符：

final：表示对象一直不变，如果修饰的是基本数据类型，表示其值一直不变，如果修饰的是引用数据类型，表示执行的引用一直不变，但是引用的对象中的值可以改变

static：静态修饰符，可以修饰变量、方法、类一旦经过修饰，就会变成类的一部分。
